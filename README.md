1. 創造repo
2. 在桌面clone
3. 把檔案放到clone檔中
4. git status檢查狀態

![](https://i.imgur.com/FTuKfeo.png)
5. git add . =>把檔案全部放到status中
![](https://i.imgur.com/878gw4S.png)
6. git status =>再次檢查狀態
![](https://i.imgur.com/K0zFm7Y.png)
7. git commit -m `"要打的commit"` =>代表版本更新數或是其他紀錄
![](https://i.imgur.com/yXr65UM.png)
8. git push =>放到github上
